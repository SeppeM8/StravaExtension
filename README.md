# Strava Speed extension

This [Chrome extension](https://chromewebstore.google.com/detail/strava-speed-kmh-mph/dmanlabgfmgkjepekbadpbgcenfbohda) allows you to see the speed of activities.

There are three options:

- Hover (Speed only visible when hovering over pace)
- Add (An extra field will be added)
- Replace (Pace will be replaced by speed)
